# Team Rocket — Nanaimo Pokémon GO Community Site

A simple static website. Plain HTML, CSS, and JavaScript. No build step.
Hosted free on GitHub Pages at **teamgorocket.com**.

## Files
- `index.html` — the whole site (one page, with sections)
- `style.css` — all the styling
- `script.js` — just the mobile menu toggle
- `CNAME` — tells GitHub Pages to serve the site at teamgorocket.com (do not delete)
- `images/` — logos, the Campfire QR code, and the ambassador photo

## How to edit text or links
1. Open `index.html`.
2. Change the words or the link (look for `href="..."`).
3. Save, commit, and push to GitHub. The live site updates within a minute.

## Swapping in higher-quality images
Replace the file in `images/` with one of the same name (e.g. a new
`logo-dark.jpg`), commit, and push. No other changes needed.

Current image names:
- `logo-dark.jpg` — logo used on dark sections (header)
- `logo-light.jpg` — logo used in the footer
- `campfire-qr.png` — the Campfire group QR code
- `ambassadors.png` — photo of Teddi & Ed
- `ambassador-program.png` — Community Ambassador Program logo
